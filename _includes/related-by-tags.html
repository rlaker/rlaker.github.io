{% comment %} Tag-aware "related posts" block. Each candidate post is scored and the highest scores are shown first: score = (number of shared tags) + (referenceBoost if this post is linked to from the current post's body) A post that the current article actually links to is a strong relevance signal, so it gets a large boost (referenceBoost) that outranks tag overlap alone. Posts linked in the body therefore float to the top, followed by the best tag matches, and any remaining slots are topped up with the most recent posts so the grid is never half-empty. Reference detection: a post is "referenced" when the rendered body contains its url (e.g. "/some-post/"). This matches both root-relative links (href="/some-post/") and full links (https://www.ronanlaker.com/some-post/). Usage: {% include related-by-tags.html %} {% endcomment %} {% assign maxRelated = 4 %} {% assign referenceBoost = 10 %} {% assign maxScore = 20 %} {% assign relatedCount = 0 %} {% assign shownUrls = "" %}
